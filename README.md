# ScoreBoard To Papi
**this feature for PlaceholderAPI to get value from scoreboard allow to show how much you have moneybank or otherthings, can work like [Tab](https://modrinth.com/project/gG7VFbG0) or other plugins.**

**adding features**
```
%scoreboard_<scoreboard name>_localplayer%
```
**or** 
```
%scoreboard_<scoreboard name>_<scoreboard username>%
```
**what mean localplayer?**

**is work like this**
```
%player_name%
```
